정규분포표에서 신뢰도가 올라가면, 유의수준이 낮아져
하지만 이 귀무가설을 기각하려면, 유의수준보다

p-value란?
------------------------------------------------------------------------
이걸 설명하려면 
유의수준, 귀무가설을 설명해야해

유의수준
- 표본을 가지고 오류를 판단할 때 오류를 범해도 감당 가능한 수준을 유의수준이라 한다. 
(0.05, 0.1, 0.01)

귀무가설
- 현재의 정설로서 관습적, 보수적 주장
새 가설에 의해 대체될 것을 예상하는 가설

제 1종 오류발생확률(p-value)
- 귀무가설이 참인데 거짓임을 주장하는 경우의 오류
(이런 일이 발생할 확률값)

그러니까 p-value가 뭔지 말로 풀어보자면,
현재 존재하는 뭐 우주의 정체에 대해 통념적으로 참으로 인정받는 가설(귀무가설)에 대해 반하는 새로운 가설이 등장햇어 근데 그 가설이 개 구라야 이 가설이 구라일 확률
------------------------------------------------------------------------
이건 좀 오류가 잇다 지피티한테 물어보니까
좋아, 지금 네가 헷갈리는 포인트를 풀어서 설명해줄게.
핵심은 p-값이 "원래 가설(null hypothesis)"이 맞다는 전제 하에서, 관측값이 얼마나 극단적인지를 나타내는 확률이라는 거야.

1. p-값의 정의
정확한 정의:
p-value = "귀무가설(H₀)이 참일 때, 현재 얻은 데이터(또는 더 극단적인 데이터)가 나타날 확률"

중요:
p-값이 H₀가 맞을 확률이 아니라는 거!
p=0.5면 "H₀가 맞을 확률이 50%"라는 뜻이 아니라,
**H₀가 맞다고 가정했을 때 이런 데이터가 나올 가능성이 50%**라는 뜻이야.

2. 네 예시 적용 — 동전 던지기
H₀: 동전은 공정하다 (앞면 확률 p = 0.5)

100번 던져서 앞면이 53번 나왔다고 하자.

53/100 = 0.53

이 정도 차이는 샘플 변동성에 의해 꽤 자주 일어남 → p-값이 높음(예: 0.5 근처)

결론: "우연히 나올 수 있는 수준이네, 굳이 가설을 버릴 필요 없다"

같은 상황에서 10,000번 던져서 앞면이 5,300번 나왔다면?

0.53이라는 차이가 이제 표본 오차에 비해 훨씬 커짐 → p-값이 매우 작음(0.001 이하)

결론: "H₀가 맞다면 이런 결과는 거의 안 나온다 → H₀를 기각하자"

3. 오해 방지
p-값 높음 (예: 0.5) → "H₀가 맞다"가 아니라 "H₀ 하에서 이런 결과가 흔하다"

p-값 낮음 (예: 0.001) → "H₀가 틀리다"가 아니라 "H₀ 하에서 이런 결과가 매우 희귀하다"
→ 그래서 H₀를 기각하는 쪽으로 간다.

4. 시행 횟수와 p-값
시행 횟수가 늘어나면 표본 평균의 분포 폭(표준오차)이 줄어듦
→ 작은 차이도 통계적으로 유의하게(p-값 작게) 나올 수 있음

그래서 대규모 표본에서는 아주 미세한 차이도 p-값이 작아져서 "유의"로 나올 수 있음
(이건 통계에서 꽤 중요한 함정)
------------------------------------------------------------------------
카이제곱 검정(chi-square test)
- 관찰된 값과 기대값의 차이를 분석하는 방법

카이제곱 = 시그마 (기대값 - 관찰값)제곱 / 기대값
이하 지피티
원리는 간단해요:

기대도수(E): 귀무가설이 참이라면 나와야 할 값
관측도수(O): 실제로 나온 값
두 값의 차이를
𝜒2=∑(𝑂−𝐸)2/𝐸
χ2 =∑E(O−E)2
로 계산합니다.

4. 이 값이 클수록 “차이가 크다 → 귀무가설이 의심스럽다”가 됩니다.

------------------------------------------------------------------------
p-값, 귀무가설, 신뢰도 정리
1. 기본 개념
귀무가설(H₀) : 기존 가설
예) 한우 1인분은 150g이다.

대립가설(H₁) : 기존 가설과 다른 가설
예) 한우 1인분은 150g이 아니다.

유의수준(α) : 허용 가능한 오류 확률
예) α = 0.05 → 5% 정도 잘못 기각 가능

p-값 : 관측 결과가 H₀ 하에서 나올 확률

판단 기준

p > α → 귀무가설 채택(기각 X)

p ≤ α → 귀무가설 기각 → 대립가설 채택

2. 예시 1: α = 0.05, p = 0.07
p > 0.05 → 관측값이 충분히 극단적이지 않음 → 귀무가설 유지

주의: 이때 “대립가설이 틀렸다”는 뜻은 아님. 단지 데이터가 H₀를 기각할 정도로 극단적이지 않다는 의미

3. 예시 2: 신뢰도 99% → α = 0.01
α = 0.01 → 기각 기준이 더 엄격

동일한 관측값에서 p = 0.01이면:

90% 신뢰도(α = 0.10) 기준에서는 기각 가능

99% 신뢰도(α = 0.01) 기준에서는 기각 경계 상황

즉, 신뢰도가 높아지면 귀무가설 기각이 더 어려워지고, p-값이 α보다 작아야 기각 가능

p-값 자체가 낮아지는 것이 아님. 기각 기준 α가 낮아져서 판단이 달라지는 것

4. 핵심 요점
p-값 : 데이터에서 계산 → H₀가 맞다면 결과가 얼마나 극단적인지

신뢰도/유의수준 α : 연구자가 정함 → H₀ 기각 기준

p-값이 작다고 무조건 H₀ 기각 아님 → α와 비교해야 결정

동전 던지기 예시

표본 크기가 크면 같은 비율 차이도 p-값 작음 → H₀ 기각 가능

신뢰도가 높아 α가 낮으면 p-값이 같아도 H₀ 채택 가능
→ 기각 여부는 p-값과 α 비교로 결정됨
